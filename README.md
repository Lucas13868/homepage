# Homepage

### Sobre o projeto
- Site de portifólio para apresentação e também para estudo de HTML, CSS (puro) e JavaScript;
- Feito como atividade do curso CS50x de Harvard;
- Site funcional tanto em computadores quanto celulares e diferentes navegadores.

### Conceitos utilizados
- Navbar;
- Popup (para configurações);
- Personalização do site pelo usuário (mudança de tema, fonte e tamanho da fonte);
- Local Storage com JS para salvar preferências de estilo do usuário;
- Elementos de HTML reutilizáveis usando class do JS;
- @media do CSS para ajustar o site da tela de computador para celular;
- Uso de grid e flexbox do CSS.
